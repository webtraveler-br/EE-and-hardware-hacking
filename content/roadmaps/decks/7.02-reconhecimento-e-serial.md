# Roadmap do Deck 7.02-reconhecimento-e-serial · Reconhecimento e Serial

**Cards:** 23

## Missão do deck
Executar reconhecimento técnico de alvo e abrir caminho para extração de dados via interfaces seriais.

## Tópicos estudados
- OSINT para IoT: buscas FCC ID, firmware público, datasheets
- Análise visual de PCB: identidade de chips, componentes passivos, interfaces
- Medição de tensão de operação: VCC pinos, GND referência, detecção esquema
- Pinagem UART: TX oscila no boot vs. RX sempre alto, GND baixo, VCC positivo
- Descoberta de baud rate: cronometro duração de bits, conversão inversa
- Protocolo SPI: pinout SOIC-8 MOSI/MISO/CLK/CS, leitura de chip com opcode 0x03
- Protocolo I2C: endereço 0x50-0x57 típico para EEPROM, i2cdetect, read word
- Análise de placas: detectar bootloader (U-Boot em regras de 20k), firmware offset
- Chain completa: UART → shell → flash dump → filesystem → analysis
- Ferramentas: Binwalk, Strings, hexdump, Ghidra, URH, nmap, i2cdetect

## Sequência recomendada
1. Revisar os conceitos-base e resolver cards novos em blocos curtos de 20 a 30 minutos.

## Projetos conectados ao deck
1. **Root Shell via UART em Dispositivo Real**
   - Escopo: Obtenha shell em dispositivo real via UART. Documente o fluxo completo: inspeção visual → identificação de pinos → descoberta de baud rate → conexão → acesso U-Boot/Linux. Portfolio: write-up com fotos de cada etapa e log de sessão.
   - Evidência: `content/roadmaps/projects/7.02-reconhecimento-e-serial-01.md`
2. **Auto Baud Detector com ESP32**
   - Escopo: Escreva firmware para ESP32/Arduino que detecta baud rate automaticamente amostrando pino TX desconhecido. Portfolio: código, testes com baud rates diferentes e documentação de uso como ferramenta de lab.
   - Evidência: `content/roadmaps/projects/7.02-reconhecimento-e-serial-02.md`
3. **Pipeline OSINT por FCC ID**
   - Escopo: Script que recebe FCC ID e puxa automaticamente fotos internas, relatórios de teste RF e diagramas de bloco. Portfolio: ferramenta funcional com exemplos de dossiês gerados.
   - Evidência: `content/roadmaps/projects/7.02-reconhecimento-e-serial-03.md`
