# Roadmap do Deck 3.02-embarcados · Embarcados

**Cards:** 45

## Missão do deck
Conectar hardware e software em firmware confiável para leitura de sensores e controle de atuadores.

## Tópicos estudados
- Arquitetura de microcontrolador: memória (Flash, SRAM, EEPROM), periféricos, pinagem
- GPIO: entrada (INPUT, INPUT_PULLUP), saída (OUTPUT); timing digital
- ADC (Analog-to-Digital Converter): 10-bit, AREF, conversão V → código numérico
- PWM (Pulse Width Modulation): duty cycle, período, aplicação em controle de luminosidade/motor
- Serial (UART): 8N1, baud rate comum (9600, 115200); TX/RX
- I2C: wire SDA - SCL, endereço 7-bit, pull-up internal; I2C scanner
- SPI: MOSI, MISO, SCK, CS; modo 0-3 (CPOL, CPHA)
- Interrupções (INT0, INT1): ISR, volatil, debounce
- Timer: millis(), micros(), delay(), setup de frequência
- Bibliotecas Arduino: sensores (LM35, LDR), atuadores (LCD, servo), protocolos

## Sequência recomendada
1. Revisar os conceitos-base e resolver cards novos em blocos curtos de 20 a 30 minutos.
2. Resolver exercícios de aplicação imediata para transformar fórmula em tomada de decisão.
3. Executar os projetos propostos e registrar evidências no diretorio de projetos da trilha.
4. Fechar ciclo com revisão FSRS diária até estabilizar acurácia acima de 85%.

## Projetos conectados ao deck
1. **Bus Pirate Lite DIY com ESP32**
   - Escopo: Firmware para ESP32 que fala UART, I2C e SPI controlado via terminal serial. Seu próprio analisador de protocolo para bancada de hardware hacking. Portfolio: firmware publicável, documentação de uso e testes com dispositivos reais.
   - Evidência: `content/roadmaps/projects/3.02-embarcados-01.md`
2. **Datalogger de Bancada com SD Card**
   - Escopo: Arduino + SD card + sensores (temperatura, tensão). Logue dados por horas, plote em Python com timestamps. Portfolio: hardware, firmware, script de plot e dados reais coletados.
   - Evidência: `content/roadmaps/projects/3.02-embarcados-02.md`
3. **Synth de Áudio PWM Bare-Metal**
   - Escopo: Gere tons musicais com timer PWM em nível de registrador (sem biblioteca Arduino). Toque uma melodia. Portfolio: código bare-metal comentado, explicação dos registradores e gravação de áudio.
   - Evidência: `content/roadmaps/projects/3.02-embarcados-03.md`
4. **I2C Device Scanner e Identificador**
   - Escopo: Firmware que varre bus I2C, identifica endereços conhecidos (EEPROM, sensor, RTC) e lê registradores de device ID quando possível. Ferramenta de reconhecimento para hardware hacking. Portfolio: banco de endereços suportados e log de dispositivos reais escaneados.
   - Evidência: `content/roadmaps/projects/3.02-embarcados-04.md`

## Critério de conclusão
- Completar 100% dos cards do deck ao menos uma vez.
- Entregar pelo menos 3 dos 4 projetos com evidência técnica clara.
- Documentar erros recorrentes e ajustes no próprio roadmap do projeto.
